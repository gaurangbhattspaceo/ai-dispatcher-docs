## The Four Headline Numbers

At the top of the page you'll see four cards:

| Metric | What It Means |
|---|---|
| **Total jobs** | Every job currently in the system (pending, assigned, and completed) |
| **Unplanned jobs** | Jobs that haven't been assigned to a technician yet |
| **Active technicians** | Technicians who are currently marked as active and available |
| **Planning rate** | Percentage of jobs that have been assigned — your "coverage" metric |

Each card also shows a trend badge (e.g. **+12%**) compared to the previous period. A rising Planning Rate and a falling Unplanned count means your dispatching is in good shape.

## Date Range and Chart Controls

Three dropdowns in the "Your overview" section control how the trend charts are displayed:

### Date Range

Changes the time window the metrics and charts cover. Options:
- **Last 7 days** (default)
- **Last 30 days**
- **Last 90 days**
- **This month**
- **Last month**
- **Custom** — pick any start and end date using the date picker

### Period

Controls the granularity of chart data points:

| Period | Data points shown |
|--------|------------------|
| **Hourly** | One data point per hour |
| **Daily** (default) | One data point per day |
| **Weekly** | One data point per week |
| **Monthly** | One data point per month |

### Comparison

Shows a comparison line on each chart so you can see how current performance stacks up:

| Option | What It Compares Against |
|--------|-------------------------|
| **Previous period** (default) | The same length of time immediately before the selected range |
| **Previous year** | The same date range from one year ago |
| **Custom period** | A date range you specify manually |
