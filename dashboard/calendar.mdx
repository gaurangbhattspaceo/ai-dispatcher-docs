# Calendar

The calendar provides a visual timeline of all technician schedules, job assignments, and availability.

## Views

- **Day View** — Detailed hour-by-hour schedule for all technicians
- **Week View** — Weekly overview with job blocks

- AI suggestion indicators showing recommended time slots
- Conflict detection highlights overlapping assignments
- Travel time estimates between jobs
