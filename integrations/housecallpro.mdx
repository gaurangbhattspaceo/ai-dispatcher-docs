This page shows you how to connect your HouseCall Pro account to AI Dispatcher. HouseCall Pro uses a **personal API key** connection — you paste your key directly into AI Dispatcher to link your accounts.

  <img src="/images/screenshots/connectors-page.png" alt="Connectors settings page" />
</Frame>

## Before You Start

You need:
- A HouseCall Pro account with admin access
- Your personal API key from HouseCall Pro (see steps below)

No OAuth or external authorization popups are needed.

## How to Find Your HouseCall Pro API Key

<Steps>
  <Step title="Open API & Webhooks in HouseCall Pro">
    In HouseCall Pro, go to **Settings → API & Webhooks**.
  </Step>
  <Step title="Copy your personal API key">
    Copy the long string shown under **Personal API Key**. It looks like a random token (letters, numbers, and dashes).
  </Step>
</Steps>

<Tip>
If you don't see an API key listed, your HouseCall Pro plan may not include API access. Contact HouseCall Pro support to enable it.
</Tip>

## How to Connect HouseCall Pro

<Steps>
  <Step title="Open Connectors">
    In AI Dispatcher, go to **Settings → Connectors** in the left sidebar.
  </Step>
  <Step title="Click Connect HouseCall Pro">
    Find the HouseCall Pro tile and click **Connect HouseCall Pro**.
  </Step>
  <Step title="Paste your API key">
    In the dialog that appears, paste your HouseCall Pro personal API key and click **Connect**.
  </Step>
  <Step title="You're connected">
    AI Dispatcher validates your key and completes the connection. HouseCall Pro begins syncing your jobs, customers, and technician availability immediately.
  </Step>
</Steps>

<Tip>
After connecting, check the **Jobs** page — your open HouseCall Pro jobs should appear there right away. If you don't see them, click **Sync from HouseCall Pro** to trigger a manual import.
</Tip>

## What Syncs Automatically

| What | Direction | When |
|------|-----------|------|
| Jobs and estimates | HouseCall Pro → AI Dispatcher | Instantly via webhook |
| Customer profiles | HouseCall Pro → AI Dispatcher | On sync |
| Technician availability | HouseCall Pro → AI Dispatcher | On connect + on demand |
| Accepted dispatch assignments | AI Dispatcher → HouseCall Pro | When you approve a suggestion |

### Real-Time Job Sync (Webhooks)

HouseCall Pro notifies AI Dispatcher whenever a job is created or changed. Your Jobs queue updates automatically in real time — no manual refreshes needed.

### Write-Back on Dispatch

Accepting an AI dispatch suggestion automatically updates the job in HouseCall Pro with the assigned technician. Your HouseCall Pro schedule stays current without any extra steps.

<Note>
Only **accepted** suggestions write back to HouseCall Pro. Rejecting a suggestion has no effect on your HouseCall Pro data.
</Note>

## Syncing Manually

To pull in the latest jobs on demand:

1. Go to the **Jobs** page
2. Click **Sync from HouseCall Pro** in the top right

## Disconnecting HouseCall Pro

Go to **Settings → Connectors**, find HouseCall Pro, and click **Disconnect**. This revokes the access token and stops syncing. Your HouseCall Pro account is not affected.

## Common Questions

<AccordionGroup>
  <Accordion title="The connection failed with an invalid API key error">
    Make sure you copied the full key — it should be a long string with letters, numbers, and dashes. The key must come from HouseCall Pro → Settings → API & Webhooks (not a different settings area). If you're unsure, generate a new key in HouseCall Pro and use that.
  </Accordion>
  <Accordion title="My jobs aren't appearing after connecting">
    Go to the Jobs page and click **Sync from HouseCall Pro** to force a manual import. Jobs must be in an open or scheduled state in HouseCall Pro to appear in AI Dispatcher.
  </Accordion>
  <Accordion title="Will disconnecting delete my jobs from AI Dispatcher?">
    No. Disconnecting only stops future syncing. Any jobs already imported into AI Dispatcher remain in the system.
  </Accordion>
  <Accordion title="Can I connect HouseCall Pro alongside another integration?">
    Each organization can have one active integration at a time. To switch from another platform to HouseCall Pro, disconnect the current integration first, then connect HouseCall Pro.
  </Accordion>
  <Accordion title="I previously connected via an authorization popup — do I need to reconnect?">
    If your connection is working, no action is needed. If you're asked to reconnect, use the API key method described above instead of the old OAuth popup.
  </Accordion>
</AccordionGroup>
